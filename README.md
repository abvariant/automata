# automata
Automata regex tutorial from gamedev.net in C++.

## Usage <BR>
``./regex2nfa "[REGEX]" "`cat [FILE TO PARSE]`"`` <BR>
<BR>
eg:<BR>
``./regex2nfa "(a|b)*abb" "`cat aaaabb.txt`"``

## Directories<BR>

contains offline original articles from 2003/2004.
